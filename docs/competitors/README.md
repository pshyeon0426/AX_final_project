# 경쟁 앱 벤치마킹 자료

슈퍼SOL과 비교할 경쟁 금융/뱅킹 앱의 벤치마킹 자료를 정리하는 폴더입니다.

## 정리 항목 (예시)

| 앱 | 운영사 | Google Play ID | App Store ID | 비고 |
|----|--------|----------------|--------------|------|
| (예) KB스타뱅킹 | KB국민은행 | `com.kbstar.kbbank` | - | |
| (예) 토스 | 비바리퍼블리카 | `viva.republica.toss` | - | |
| (예) 카카오뱅크 | 카카오뱅크 | `com.kakaobank.channel` | - | |

## 분석 관점

- 핵심 기능 비교 (송금, 자산조회, 혜택 등)
- 사용자 평점 / 리뷰 감성 추이
- 주요 불만 / 칭찬 키워드
- UX / 온보딩 흐름

> 각 경쟁사별 상세 자료는 `docs/competitors/<앱이름>.md` 로 분리하여 작성합니다.
