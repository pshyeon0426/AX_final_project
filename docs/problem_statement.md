# 문제 정의서 (1p) — 슈퍼SOL 고객 반응 분석 서비스

신한은행 AX 전문가 Intensive 최종 프로젝트 · 4조(댓글부대)

## 배경
슈퍼SOL **출시 초기**에는 고객 반응을 빠르고 일관되게 파악하고, 경쟁 금융 앱 대비 위치를
진단하며, 부정 리뷰에 신속히 응대하는 것이 중요하다. 그러나 수작업 모니터링은 느리고
주관적이며, 답글 작성도 사람마다 품질이 달라 비효율적이다.

## 문제
- 공개 앱 리뷰(1차 VOC)가 흩어져 있어 **초기 반응·주요 불만을 정량화**하기 어렵다.
- 경쟁 앱과의 **객관적 비교 기준**이 없다.
- 부정 리뷰 **답글 작성에 시간이 많이 들고 일관성**이 떨어진다.

## 목표
공개 리뷰만으로 **① VOC 분석(감성·이슈·토픽) → ② 경쟁 벤치마킹 → ③ 답글 초안 생성**까지
하나의 대시보드로 자동화하여, CX/기획 담당자의 의사결정과 응대 생산성을 지원한다.

## 범위 / 비범위
- **범위**: 공개 스토어 리뷰 수집·분석, 경쟁 비교, 검수용 답글 초안.
- **비범위**: 답글 자동 게시(❌), 실제 고객정보·내부 비공개 데이터 사용(❌),
  경쟁사 내부 전략 추정(❌).

## 핵심 KPI / 성공 기준
- 데이터 품질 검증 통과(결측 ≤5%·중복 ≤3%·총 ≥1,000건 등)
- 감성 모델 baseline 대비 macro F1 개선
- 답글 안전성 위반 0건(검수용 초안 + 금지 원칙 + 후처리 검증)
- 사전 산출물 기준 분석 응답 ≤ 30초

## 금융 업무 활용 가능성
출시 모니터링·경쟁 포지셔닝·CX 답글 생산성·개선 우선순위 도출. **내부 데이터 없이도** 즉시
활용 가능하며, 내부 데이터 결합 시 정확도·범위 확장(선택적 고도화).
